# Programacion 1 - Tipos de Datos y Entrada/Salida en C++

## Descripcion

Este repositorio contiene la tarea sobre **tipos de datos primitivos** y **entrada/salida basica** en C++ para el curso de Programacion 1 de la UNIAT.

Los ejercicios estan disenados para estudiantes de segundo semestre que estan aprendiendo los fundamentos del lenguaje C++.

## Como Usar Este Repositorio (Estudiantes)

1. Haz un **fork** de este repositorio en tu cuenta de GitHub.
2. **Clona** tu fork en tu computadora.
3. Abre la carpeta `assignment/` y lee el archivo `README.md` con las instrucciones.
4. Completa cada archivo `.cpp` en las carpetas de cada ejercicio.
5. Compila y prueba tu codigo antes de entregar.
6. Sube tus cambios con **commit** y **push** a tu fork.

Consulta `CONTRIBUTING.md` para una guia paso a paso.

## Temas que Cubre Esta Tarea

- Tipos de datos primitivos: `int`, `float`, `double`, `char`, `bool`, `string`
- Declaracion e inicializacion de variables
- Operaciones aritmeticas basicas
- Entrada de datos con `cin`
- Salida de datos con `cout`
- Estructura basica de un programa en C++
- Comentarios y buenas practicas de escritura de codigo

---

## Requisitos para Compilar

Necesitas un compilador de C++ instalado. Las opciones mas comunes son:

- **g++ (MinGW en Windows / GCC en Linux y Mac)**
- **Visual Studio** (Windows)
- **Code::Blocks**, **Dev-C++**, o cualquier IDE compatible

### Compilar con g++ desde la terminal

```bash
# Compilar un ejercicio
g++ ejercicio1.cpp -o ejercicio1

# Ejecutar el programa compilado
./ejercicio1        # Linux / Mac
ejercicio1.exe      # Windows
```
